## server启动方法
在本地调试博客
安装
$ sudo gem install jekyll -n/usr/local/bin    ‘有的路径没有权限，需要安装在这里’

sudo gem install jekyll bundler -n/usr/local/bin

sudo gem install  -n/usr/local/bin

 直接输入密码回车就可以
 
 启动服务器
 
 $jekyll s 
 显示ip 地址， 照地址输入就可以
